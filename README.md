
# ConsultPro Mobile-First Premium UI

## Setup

Create `.env.local`

NEXT_PUBLIC_SUPABASE_URL=YOUR_URL
NEXT_PUBLIC_SUPABASE_ANON_KEY=YOUR_KEY

## Install

npm install

## Run

npm run dev

## Deploy

Push to GitHub and deploy using Vercel.
